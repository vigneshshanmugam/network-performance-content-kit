Network Performance Content Kit

This Content kit provides learning material to allow Advanced Web Developers to level up thier skills, learn world class techniques on imporving the performance of thier Web Applications.

For any issues, submit issues against the network-performance-content-kit repository.

What should the presenter have?

What hardware and software will the presenter ideally have?

  • Software
    • A Linux, Windows or Mac OSX computer.
    • 1-3 modern browsers installed. This could include Firefox, Chrome, Opera, Safari, or Internet Explorer 10+.
    • A decent text editor installed. Examples include Sublime Text, Atom, or Notepad++.
  • Coding knowledge
